O QUE É O JAVA SCRIPT ?

O JavaScript é uma linguagem de programação de script que permite a criação de páginas internete.

É uma linguagem de SCRIPT de programação que roda no lado cliente (navegado do usuário) e nos permite realizar ações detro de uma pagina web, melhorando a interatividade com o usuário.

JavaScript não é Java

JavaScript não é uma linguagem compilada e sim interpretada, ou seja, para que ela seja executada, é necessário que o navegador a interprete. Dessa forma, é fundamental que o navegador seja compatível com a linguagem de script.

navegadores = Firefox, Chrome, Enternet,..

O QUE SE PODE FAZER COM O JAVASCRIPT ?

Criar páginas web interativas.

Desenvolvimento de codigos do lado do servidor e desenvolver extensões de aplicações com javascript.

Desenvolvimento multiplataforma juntamente com HTML e CSS e criar aplicações para celulares ou desktops.

Manipular elementos HTML

Vantagens de usar JavaScript com um arquivo externo

Facilidade de Manutenção

Uma vez que o JavaScript está localizado em apenas um arquivo, facilitando assim a edição ou correção dos códigos.

Carregamento mai rápido da página

O arquivo externo é armazenado no cache do navegador. Assim, evita-se carregá-lo toda vez que a página for chamada

vantagem semântica

O arquivo externo separa a camada de comportamento (javascript) da camada de conteúdo (HTML)
